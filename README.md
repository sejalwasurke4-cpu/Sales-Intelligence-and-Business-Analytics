# Sales Intelligence and Business Analytics

## 📌 Project Overview
This is an end-to-end Sales Intelligence and Business Analytics project focused on analyzing sales data to uncover trends, patterns, and actionable business insights that support data-driven decision-making.

## 🛠️ Tools & Technologies Used
- SQL – Data extraction, joins, aggregations, and business queries  
- Python (Pandas, NumPy) – Data cleaning, preprocessing, and analysis  
- Excel – Data validation, pivot tables, and summary analysis  
- Power BI – Interactive dashboards and visual storytelling  

## 📊 Key Analysis Performed
- Monthly and yearly sales trend analysis  
- Top-performing products and categories identification  
- Customer retention and churn analysis  
- Average Order Value (AOV) calculation  
- Duplicate record detection and handling  
- Missing value treatment and data preprocessing  

## 💡 Business Insights
- Identified high revenue-generating products and categories  
- Analyzed repeat customer behavior and purchase patterns  
- Highlighted low-performing months and sales dips  

## 📈 Outcome
This project helps businesses monitor sales performance, understand customer behavior, and make informed, data-driven strategic decisions.
